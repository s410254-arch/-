# BabyCare 智慧嬰兒照護系統 UI 設計

## 1. 產品定位
- App 名稱：**BabyCare 智慧嬰兒照護系統**
- 風格：現代化、簡潔、溫馨
- 色系：藍白為主，搭配柔和淺灰與溫暖橘紅作為警示色
- 調性：安全、專業、科技感、高級感

## 2. 視覺設計要素
### 2.1 主色調
- 主色：柔和靛藍 / 天空藍
- 輔色：純白、米白、淺灰
- 警示色：橘色、紅色

### 2.2 排版與圖層
- 大面積白底或淺藍卡片
- 圓角元素：卡片、按鈕、資料區塊皆採 16-24pt 圓角
- 平滑陰影：低飽和、柔和擴散陰影，營造浮起效果
- 空間感：留白充足，資訊區塊之間保持視覺呼吸感

### 2.3 字體與文字層級
- 標題：清晰大字體，中英混排友好
- 內文：簡潔、易讀字重
- 標籤：使用色塊或輕量底色標示

### 2.4 Icon 與圖示
- 圖示風格：線性 + 實心填充混合，簡潔直覺
- 監測數據圖示：溫度計、滴水、嬰兒翻身、燈泡、警報鈴
- 轉換狀態提示：正常為柔和藍綠，異常為橘/紅

## 3. 主頁（Dashboard）
### 3.1 頂部區塊
- 左側：BabyCare Logo，搭配簡潔識別字樣
- 右側：系統連線狀態
  - 圓形指示燈 + 文字
  - 例如：「已連線」「離線」「訊號穩定」

### 3.2 主要資訊卡
- 卡片式設計，採用圓角白底與柔和陰影
- 內容分成主要監測區與警報摘要

#### 3.2.1 即時監測數據
- 溫度：數位儀表＋溫度計圖示
  - 中央呈現大字體攝氏數值
  - 下方附屬文字：目前舒適區間、趨勢箭頭
- 濕度：數位儀表＋水滴圖示
  - 顯示相對濕度百分比
  - 附註當前濕度區間

#### 3.2.2 嬰兒翻身狀態
- 使用嬰兒圖示搭配簡短文字
- 正常：綠色或藍色文字「翻身正常」
- 異常：橘色/紅色圖示與「翻身頻率異常」或「請注意」

#### 3.2.3 房間燈光狀態
- 燈泡圖示，顯示「開燈 / 關燈」或「微光模式」
- 若有自動情境，可補述目前模式

#### 3.2.4 系統警報資訊
- 以卡片底部警報區塊展示
- 異常時使用橘色或紅色背景塊
- 文字示例：
  - 「溫度過高，建議降溫」
  - 「濕度異常，請檢查空調」
- 正常時顯示安全提示：「目前一切正常」

### 3.3 底部導航
- 四個分頁按鈕：首頁、數據分析、警報紀錄、系統設定
- icon + 文字標籤
- 當前分頁採用藍色底或底下橫線高亮

## 4. 數據分析頁
- 版面核心：歷史資料趨勢視覺化
- 使用淺色卡片、圓角邊框、柔和陰影

### 4.1 圖表內容
- 溫度折線圖：展示過去 24 小時 / 7 天趨勢
- 濕度折線圖：同樣展示時間趨勢
- 翻身次數折線圖：每小時間隔數值

### 4.2 統計摘要
- 卡片式統計區塊：平均值、最高、最低、警報次數
- 使用清楚數字與小圖示
- 提示資訊：例如「上週平均 24.2°C，穩定」

### 4.3 互動細節
- 滑動時間範圍選單：24H / 7D / 30D
- 圖表點按可顯示單點數據細節

## 5. 警報紀錄頁
- 以時間軸方式展示
- 每條事件採用卡片或時間節點

### 5.1 事件類型
- 溫度過高 / 過低
- 濕度異常
- 翻身頻率過高或過低
- 連線中斷

### 5.2 顯示內容
- 時間戳記、警報等級、簡短描述
- 若可點選：顯示對應處理建議、事件詳情

### 5.3 視覺風格
- 正常事件使用柔和灰底
- 異常事件使用橘色或紅色亮點
- 時間軸左側圓點 + 右側連線

## 6. 系統設定頁
- 採卡片式設定區
- 清晰分區：警報門檻、Wi-Fi 連線、通知設定

### 6.1 警報門檻值
- 溫度上下限設定
- 濕度上下限設定
- 翻身警示頻率上限/下限
- 採用切換開關 + 數字輸入 / 滑桿

### 6.2 Wi-Fi 連線資訊
- 顯示目前 SSID、連線狀態
- 支援切換網路、重新連線、Wi-Fi 設定
- 可顯示訊號強度、AP 名稱

### 6.3 通知方式
- 推播通知開關
- 聲音/震動選項
- 緊急警報模式設定

## 7. 介面體驗建議
- iOS / Android 一致設計語言，但保留平台原生觸感
- 使用大按鈕與可讀字級，符合嬰兒照護產品的親切度
- 避免過度資訊，同時讓重要警報在第一時間可辨識
- 產品化細節：底部導航穩定、卡片分區明確、圖表精緻

## 8. 風格總結
- BabyCare 的首頁應呈現「安全穩定、科技監控、貼心關懷」
- 圓角卡片與柔和陰影打造現代質感
- 藍白主色系強調專業性與清爽感
- 橘紅警示色則維持使用者注意力，避免誤判

## 9. 建議延伸
- 未來可加入「寶寶心跳監測」、「夜間音量分析」、「雲端報告」
- 可擴充為智慧嬰兒房家長模式，支援多台監控設備
